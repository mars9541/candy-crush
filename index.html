Shape Match 3 is a match 3 game, using Phaser game engine and HTML5 specification

Not finished yet, but playable😆~~

How to play

Swap 2 shapes with mouse or finger, let 3 same shapes connect into a line, you can eliminate

Play now!

This game supports offline play, if you feel that the game has not been updated, you can pressHereForce update

Can't play the game?

If the game screen keeps flickering, the screen is messed up, or the screen is blank, it may be because your device does not support hardware acceleration

You can entersetting，change renderer to CANVAS

If you only see "Unable to load Phaser", then I guess you are using a very old browser. This doesn't support versions below IE 9, please use Chrome or Firefox instead

If stuck in "Loading assets...", it may be

  1. The network is not strong
  2. Your browser cannot decode the sound

Whatever the reason, you can entersetting，Tick Disable audio
